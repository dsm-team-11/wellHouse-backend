# 위험도 판정 엔진

`domain/risk/RiskEngine.java` · 임계값은 `domain/risk/Thresholds.java`.

## 최종 위험도

```
최종 = MAX(수위단계, 강수단계, 특보floor, 상승속도단계, 예보floor)
       그다음 +1 상향(bump): 상승속도>=2cm/min, 최근3h누적>=60mm
       마지막에 DANGER(3)로 클램프
```

### 서브 판정

| 요소 | 규칙 |
| --- | --- |
| 수위(cm) | `<3` 양호 · `3~10` 주의(정지) / 경고(지속상승 ≥0.5cm/min) · `≥10` 위험 |
| 강수(mm/h) | `<15` 양호 · `15~30` 주의 · `30~50` 경고 · `≥50` 위험 (`≥80` 극한호우 플래그) |
| 특보 | 호우주의보→최소 주의 · 호우경보→최소 경고 (해제 전 하향 금지) |
| 상승속도(cm/min) | `≤0.5` 양호 · `0.5~2` 주의 · `≥2` 경고 + 최종 **+1** |
| 누적강수 | 최근 3h `≥60mm` → 최종 **+1** |
| 예보 | 2h내 `≥30` 주의 · `≥50` 경고 (선행 대응) |

## 상태 전이 (`domain/risk/StateMachine.java`)

- **승격**: 즉시.
- **강등**: 한 번에 한 단계씩, 안정 유지 시간 필요.
  - 위험→경고 10분 · 경고→주의 20분 · 주의→양호 30분
- 특보 floor 아래로는 절대 강등하지 않음.
- 구현: `state.candidateSince` 에 "현 단계 미만이 연속 유지되기 시작한 시각"을 저장,
  경과 시간이 임계 도달 시 한 단계 내리고 타이머 리셋.

## 골든타임 (`domain/risk/GoldenTime.java`)

```
v = 상승속도(m/s)          # 실측 cm/min 을 환산
T = (h_target - h_current) / v
```

- 상승하지 않으면(v≤0) `reachable:false`, `seconds:null`.
- 인체 기준 ETA 동시 제공: 무릎0.4 · 허리0.9 · 가슴1.2 · 턱1.5 · 코1.6 (m).

## 자동 조치 (승격 시)

| 단계 | 조치 |
| --- | --- |
| 주의 | 체크리스트/배수구 점검 알림 |
| 경고 | 물막이판 전개 + 수도밸브 차단 + 두꺼비집 차단 명령, 알림 |
| 위험 | 비상모드, 30초 자동신고 카운트다운, 대피 안내 |
